TicTacTic

TicTacTic is a variation of the classical TicTacToe or Tris game, made more interesting by the introduction of the concept of the "Aging cells", to avoid the sure draw of TicTacToe game: each player can occupy just 3 cells, when one more cell is occupied, the oldest one is freed. The manual is included in the package

The game has been implemented in Caml, using Tk as graphic toolkit. It's freely available under GPL, and it comes with precompiled binaries, both for Linux and Windows. The program is given under the GPL License.

The software is available as Windows executable (compiled in year 2000) or OCaml sources (written in year 2000).
